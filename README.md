# 🌽 Corn & Culture Club

A weekly local newsletter + website for parents of kids (newborn–high school) in Johnson
County, Iowa — Iowa City, Coralville, North Liberty, Solon, Tiffin, and beyond.

One email a week: what's worth doing with your kids, what's free, what signup deadline
you're about to miss, and one cool local thing you didn't know about.

Inspired by [Catskill Crew](https://catskillcrew.com/), built to be as automated as
possible with Claude Code skills (research → write → preview → publish).

**Start here: [PLAN.md](PLAN.md)** — the full build plan, content buckets, source map,
architecture, and phased roadmap.
